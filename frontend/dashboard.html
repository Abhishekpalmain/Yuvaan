How It Works

1. GST Reconciliation (Anti-Fraud)

The system compares the Input Tax Credit (ITC) claimed by the borrower in GSTR-3B against the actual supplier invoices in GSTR-2A. Any over-claim indicates potential tax evasion or balance sheet inflation.

Demo finding: TechForge claimed ₹300,000 IGST but suppliers only filed ₹414,000 in invoices? Actually they overclaimed by ₹114,000!

2. Circular Trading Detection (Graph Analytics)

Bank transactions are modeled as a directed graph. The system detects closed loops where money circulates between related entities, returning to the origin. This is a classic fraud pattern to inflate bank balances artificially.

Demo finding: TechForge → ShellCorp Consultants → ShellCorp Holdings → TechForge = Circular loop detected!

3. Explainable AI (XAI)

Unlike black-box models, Intelli-Credit documents every score adjustment with explicit reasoning. Regulators (RBI) require this transparency. Each deduction includes:

  • Exact point value
  • Source transaction/evidence
  • Risk category
